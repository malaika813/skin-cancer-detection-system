About the Skin Cancer Detection System

This AI-powered web platform helps detect skin cancer using a deep learning model. It lets users upload images of skin lesions to instantly receive predictions about possible cancer types, empowering early diagnosis.

Vision & Mission

Our mission is to make early detection of skin cancer accessible and faster for everyone. We aim to support healthcare providers through AI-powered screening tools and reduce late-stage diagnoses by enabling early awareness and action.

Motivation

Skin cancer is one of the most common yet preventable cancers globally. Early diagnosis can significantly improve survival rates. With the power of AI, we can analyze images more quickly and accurately than traditional methods, saving time and lives.

Dataset Used

We used the HAM10000 dataset — a large collection of over 10,000 images representing various skin lesion classes such as Melanoma, BCC, AKIEC, and more. These high-quality labeled images helped train our CNN model for accurate predictions.

How It Works

The user uploads an image of a skin lesion, which is then preprocessed and passed through a trained Convolutional Neural Network (CNN). The model analyzes the image and returns the predicted condition along with the result stored in the user’s history.

Core Features

Key features include user login/signup authentication, uploading and predicting skin conditions from images, viewing past predictions, and accessing educational resources on skin cancer.

Technical Implementation

This platform is developed using Python and Flask for the backend. The prediction system uses machine learning models (CNN) trained on medical image datasets. The frontend is fully responsive and built with Bootstrap 5 to support seamless use on all devices.
